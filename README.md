# 知乎 Feed 过滤器

一个 Chrome 浏览器扩展，自动屏蔽知乎信息流中的广告和关键词内容。

## 功能

- **广告自动屏蔽** — 自动识别并隐藏信息流中的推广/赞助内容
- **关键词过滤** — 自定义关键词，标题或摘要命中即隐藏
- **实时生效** — 规则变更后立即更新，无需刷新页面
- **多设备同步** — 规则通过 Chrome 账号同步到所有设备

## 安装

### 第一步：下载

点击页面右上角绿色 **Code** 按钮 → **Download ZIP**，解压到任意目录。

### 第二步：加载到 Chrome

1. 打开 Chrome，地址栏输入 `chrome://extensions` 并回车
2. 开启右上角「**开发者模式**」
3. 点击「**加载已解压的扩展程序**」
4. 选择刚才解压的文件夹

完成后工具栏会出现扩展图标。

> 由于未上架 Chrome 商店，每次启动 Chrome 可能会弹出「停用开发者模式扩展程序」的提示，点击「取消」即可正常使用。

## 使用

点击 Chrome 工具栏中的扩展图标打开控制面板：

- **启用过滤** — 总开关，关闭后所有被隐藏的内容恢复显示
- **自动屏蔽广告** — 开启后自动识别并隐藏广告内容
- **关键词列表** — 添加/删除自定义过滤关键词（最多 100 个，每个最多 100 字符）

```
├── manifest.json       # Chrome 扩展清单 (Manifest V3)
├── content.js          # 注入知乎页面的过滤脚本
├── popup/
│   ├── popup.html      # 弹窗界面
│   ├── popup.js        # 弹窗逻辑
│   └── popup.css       # 弹窗样式
├── icons/              # 扩展图标
└── tests/              # 单元测试
```

## 技术说明

- 使用 `MutationObserver` 监听知乎 React 根节点，实时处理新增内容
- 命中规则的节点设置 `display: none` 并打上 `data-zf-filtered` 标记
- 规则存储于 `chrome.storage.sync`

> 广告识别基于知乎当前 DOM 结构，若知乎前端更新可能需同步维护选择器。
